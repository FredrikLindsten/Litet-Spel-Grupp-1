The coordinate system starts from the bottom left corner of the map.

Först allt levelspecifikt såsom flaggor, detaljer, osv
//Startpos:(int, int) coordinate, Endpos:(int, int) coordinate


Levelformatet består därefter av en lista med GameObjects.
(int, int) coordinate, (4x4 float) transformation matrix, (int) render object reference, (int) game object type reference

GameObject-typ Flaggor:
	Dörr: (bool) öppen/stängd, (int) 0 = normal dörr, 1 = startdörr, 2 = slutdörr, (int) activation ID
	Spak: (bool) power on/off, (int) activation ID [om ej: -1], (bool) activated on/off, (int) activates object reference ID
	Pressure plate: (int) activates object reference ID
	Shadow container: (int) activates object reference ID
	
	
**I spakens fall, står power-bool för om möjlighet ATT aktivera spaken finns. Activated-boolen beskriver om den använts för att aktivera övrigt gameobject.
